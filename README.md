# tabc-program
Sunday Service Flow

Truth Alone Bible Church — Worship Service Program Builder
A single-file mobile-friendly web app that generates the Sunday worship program as a JPG image, styled to match your existing program template.
What it does

Auto-fills the upcoming Sunday's date (today if today is Sunday)
Lets you tick which items will be present and edit the time for each
Lets you add custom items (pre-service, sub-items, post-service)
Lets you enter sermon details (preacher, title, verse) manually
Shows a live preview that updates as you edit
"Done" button generates a 1080×1920 JPG and:

On mobile: opens the system share sheet (share straight to WhatsApp)
On desktop or older browsers: downloads the JPG
